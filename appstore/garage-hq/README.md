# Garage HQ

Garage 是一种与 S3 兼容的分布式对象存储服务，专为中小规模的自托管而设计。

## 简介

Garage 专为由运行在不同物理位置的节点组成的存储集群而设计，旨在轻松提供存储服务，该服务可在这些不同位置复制数据，即使某些服务器无法访问也能保持可用。Garage
还注重轻量级、易于操作以及对机器故障的高弹性。
